# T-164「顏色獵人」反應遊戲

由 T-161「67 獵人」（本身 fork 自 T-152「數字獵人」）改版而來的純前端單檔遊戲（`index.html`，雙擊即玩、零相依、手機優先）。

## 玩法
棋盤（A4 隨機 Voronoi 格）上每一格都有**底色**與一個**顏色文字**。
- **多數格相符**：底色＝文字（例：黃底寫「黃」、藍底寫「藍」）。
- **少數「目標格」對不上**：文字＝本局目標顏色詞，但底色是別的顏色（例：底是綠色、字卻寫「藍」）。
- **每局隨機選一個目標顏色詞**，遊戲**語音會唸出它**（HUD 也顯示「找出 → 藍」）。
- 玩家要用**最快速度點出所有「文字＝目標詞但底色不是該色」的格**就獲勝；計時到 0.01 秒，越快排名越前。

陷阱設計：盤面也會放幾格「文字＝目標詞、底色也＝該色」的**相符格**（例：藍底寫「藍」），逼玩家**看底色而非只讀字**。
- 點到相符格（底色＝文字）→ 扣 1 條命，錯 3 次遊戲結束（不計分）。
- 找到全部目標格 → 過關，留名進排行榜（依用時由小到大排序）。
- **四種難度＝格子數**：新手 25 / 簡單 50 / 普通 75 / 困難 100；目標格數約為格數的 12%（3 / 6 / 9 / 12 個）。
- 目標格可能藏在很小的格子裡，**兩指捏合放大 / 滑鼠滾輪縮放**找；切到背景自動暫停。
- 文字字色依底色明暗自動切換（深底白字、亮底黑字），確保「文字」一定看得清——被測的是「底色」。
- 中英雙語（英文顯示英文顏色名，長字依字數縮放塞進格內）、背景音樂、音效、語音報色。

紅 Red／橙 Orange／黃 Yellow／綠 Green／藍 Blue／紫 Purple／粉 Pink／棕 Brown／灰 Gray／黑 Black。
