# T-164「顏色獵人」反應遊戲

由 T-161「67 獵人」（本身 fork 自 T-152「數字獵人」）改版而來的純前端單檔遊戲（`index.html`，雙擊即玩、零相依、手機優先）。

## 玩法（v3，2026-06-15 改版）
棋盤（螢幕隨機 Voronoi 格）上每一格**開局底色都是白的**，每格有一個**顏色文字**。
- **Stroop 干擾**：每格文字的**顏色故意跟字義不同**（例：字寫「藍」，卻用綠色描繪）。
- **語音隨機念出一個顏色詞**（HUD 也顯示「點出 → 藍」）。
- 玩家要**點出「文字寫著那個詞」的格子**（別被字的顏色騙了）；點對後，那格白底就會**補滿成跟字義一樣的顏色**（對上了、加綠框鎖定），接著**自動念下一個顏色詞**。
- 如此一路把**所有格子都對上**就過關；計時到 0.01 秒，越快排名越前。
- 點錯（文字不是目前念出的那個色）→ 扣 1 條命，錯 3 次遊戲結束（不計分）。
- **四種難度＝格子數**：新手 25 / 簡單 50 / 普通 75 / 困難 100（要對上的目標＝全部格子）。
- 小格可**兩指捏合放大 / 滑鼠滾輪縮放**找；**切到背景/失焦自動暫停**（背景音樂一律暫停，遊戲中連同計時/棋盤一起暫停）。
- 開局文字用**鮮明的描繪色**（白底上清楚可讀）；對上後填滿用**淡粉柔和色系**（淺色 → 文字深色，淺底深字清楚可讀）。
- 中英雙語（英文顯示英文顏色名，長字依字數縮放塞進格內）、背景音樂、音效、語音報色。
