# T-164「顏色獵人」反應遊戲

由 T-161「67 獵人」（本身 fork 自 T-152「數字獵人」）改版而來的純前端單檔遊戲（`index.html`，雙擊即玩、零相依、手機優先）。

## 玩法（v2，2026-06-15 改版）
棋盤（A4 隨機 Voronoi 格）上每一格都有**底色**與一個**顏色文字**。
- **開局全部對不上**：每一格的底色都跟它的文字不一樣（例：底是綠色、字卻寫「藍」），整盤都是「對不上」的。
- **語音隨機念出一個顏色詞**（HUD 也顯示「點出 → 藍」）。
- 玩家要**點出「文字寫著那個詞」的格子**；點對後，那格底色就會**補成跟文字一樣的顏色**（對上了、加綠框鎖定），接著**自動念下一個顏色詞**。
- 如此一路把**所有格子的顏色與文字都對上**就過關；計時到 0.01 秒，越快排名越前。
- 點錯（文字不是目前念出的那個色）→ 扣 1 條命，錯 3 次遊戲結束（不計分）。
- **四種難度＝格子數**：新手 25 / 簡單 50 / 普通 75 / 困難 100（要對上的目標＝全部格子）。
- 小格可**兩指捏合放大 / 滑鼠滾輪縮放**找；切到背景自動暫停。
- 配色為**淡粉柔和色系**（皆為淺色），文字一律深色（黑色詞為白字），淺底深字始終清楚可讀。
- 中英雙語（英文顯示英文顏色名，長字依字數縮放塞進格內）、背景音樂、音效、語音報色。

紅 Red／橙 Orange／黃 Yellow／綠 Green／藍 Blue／紫 Purple／粉 Pink／棕 Brown／灰 Gray／黑 Black。
